# Tic Tac Toe Game

A Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features

- Two Player Mode
- One Player Mode with Bot
- Win Detection
- Draw Detection
- Restart Game Option
- Interactive User Interface
- Responsive Design

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Play

### Two Player Mode
- Player O starts the game.
- Players take turns marking cells.
- The first player to get three marks in a row wins.

### One Player Mode
- Play against a computer bot.
- The bot automatically makes its moves.
- The game detects wins, losses, and draws.
